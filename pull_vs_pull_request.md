# Difference Between Pull, Pull Request, and Push

## Push
Uploads your local commits to the remote GitHub repository.

## Pull
Downloads and merges changes from the remote repository into your local branch.

## Pull Request
Requests that someone else review and merge your changes into their repository.

## Summary Table

| Action | Description |
|------|------------|
| Push | Send local changes to GitHub |
| Pull | Get updates from GitHub |
| Pull Request | Ask to merge changes |

## Key Difference
Push and pull are Git commands, while pull requests are a GitHub collaboration feature.
