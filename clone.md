# Clone

A **clone** is a local copy of a GitHub repository downloaded to your computer.
Cloning includes all files, branches, and commit history so you can work offline.

## Why Cloning Is Useful
- Lets you edit files locally
- Allows committing changes step by step
- Enables pushing updates back to GitHub

## Example Command
```bash
git clone https://github.com/username/repository-name.git
```
